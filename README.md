je montre comment accéder à un serveur web ESP32 directement depuis Internet grâce à l’IPv6.

Pas besoin :

* d’ouvrir un port sur la box
* de ngrok
* de cloud
* de MQTT
* de PC allumé

Fonctionnalités :

* ESP32 WebServer
* IPv6 globale
* accès distant
* login / mot de passe
* raccourci Android
* AsyncWebServer

Matériel :

* ESP32
* connexion Internet compatible IPv6
* téléphone 4G/5G

Bibliothèques utilisées :

* WiFi.h
* WebServer.h
* ESPAsyncWebServer
